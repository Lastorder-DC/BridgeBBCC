************************* 사 용 법 *************************
*                                                          *
* 1. chzzk-chat-server를 설치 및 실행                     *
*    https://github.com/Lastorder-DC/chzzk-chat-server     *
*    .env 파일에 CLIENT_ID, CLIENT_SECRET, REDIRECT_URI   *
*    를 입력한 후 npm install && npm start                 *
*                                                          *
* 2. 아래를 참고해 lib/config.js의 내용을 조정            *
*    chzzkChannelId : 구독할 채널 ID                       *
*    chzzkAuthServerUrl : chzzk-chat-server 주소           *
*                                                          *
* 3. 송출 소프트에어의 Webpage Source에 client.html을 추가 *
*                                                          *
* 4. 처음 실행 시 화면에 치지직 로그인 링크가 표시됩니다.  *
*    OBS 브라우저 소스에서는 팝업이 열리지 않으므로,       *
*    표시된 링크를 일반 브라우저에서 열어 로그인하세요.    *
*                                                          *
* 5. 로그인 후 자동으로 채팅이 연결됩니다.                 *
*    로고 표시로 로딩완료를 확인한 후 방송하기             *
*                                                          *
* 6. 로그아웃: OBS 브라우저 상호작용으로 마우스를 움직이면 *
*    우측 상단에 반투명 로그아웃 버튼이 표시됩니다.        *
*                                                          *
*                                                          *
*           API 부존재로 삭제된 기능 (v0.6.0)             *
* -------------------------------------------------------- *
* - 트위치 채팅 연동                                       *
*   (트위치 IRC 완전 제거, CHZZK으로 교체)                 *
* - 비트/치어 아이콘 표시 (loadCheerImgs)                  *
*   치지직은 비트 후원 방식이 없음                         *
* - 트위치 이모티콘/구독콘 표시 (loadTwitchCons)           *
*   치지직 공식 API에서 이모티콘 목록 미제공               *
* - 뱃지 표시 (badgeVisible, badgeChannelId)               *
*   치지직 공식 API에서 뱃지 이미지 URL 미제공             *
* - 클립 링크 별도 처리 (clipReplaceMsg)                   *
*   치지직 공식 API에서 클립 개념 미지원                   *
* - twipkr 후원 연동 (twip_apply.js)                       *
*   치지직은 twip 연동 API 미지원                          *

********************** config.js 설정 **********************
*                                                          *
* 접근성 표시 (*)를 참고                                   *
* lib\config.js의 내용을 설정할 것                         *
*                                                          *
*                                                          *
* numChatMax       (*) 한번에 표시할 최대 메세지 수        *
*                                                          *
* personalColor    (*) 이름 색을 닉네임 기반으로 자동배정  *
*                  -> true, false                          *
*                                                          *
* themeUri         (***) 테마를 불러올 주소                *
*                  "https://..."                           *
*                  빈 문자열일 경우 상대 경로에서 불러옴   *
*                                                          *
* theme            (***) 사용할 기본 테마                  *
*                  theme 폴더 안의 하위 폴더 이름 문자열   *
*                  -> "테마 이름"                          *
*                                                          *
* msgExistDuration (*) 각 메세지의 제한 시간 (초)          *
*                  -> 0 입력시 제한 시간을 두지 않음       *
*                                                          *
* msgAniDuration   (**) 메세지가 사라질 때 표시되는        *
*                  애니메이션의 동작 시간 (초)             *
*                  -> 0 입력시 애니메이션을 표시하지 않음  *
*                  -> -1 입력시 테마의 기본 값을 사용      *
*                                                          *
* debugLevel       초기 로딩 메세지를 포함한               *
*                  디버그메세지의 표시 여부                *
*                  -> 0 (미표시)                           *
*                  -> 1 (개발자도구에 로그만 남김)         *
*                  -> 2 (메세지로 출력)                    *
*                                                          *
* loadDcCons       (**) 디시콘 사용 여부                   *
*                  -> true, false                          *
*                                                          *
* dcConsURI        (**) 디시콘을 불러올 주소               *
*                  ""일 경우 pc의 하위디렉토리에서 불러옴  *
*                  lib/dccon_list.js와                     *
*                  images/dccon 폴더를 사용                *
*                                                          *
* chzzkDonationMsg (*) 치지직 후원 메세지의 강조 문구      *
*                  {amount}는 후원 금액으로 대체됨         *
*                  {0:내용}은 금액을 알 수 없을 때 표기    *
*                  {!0:내용}은 그렇지 않을 경우에만 표기됨 *
*                                                          *
* chzzkSubMsg      (*) 치지직 구독 기념 메세지의 강조 문구 *
*                  {months}는 개월수로 대체됨              *
*                  {0:내용}은 구독 시작시에만 표기         *
*                  {!0:내용}은 구독 유지시에만 표기됨      *
*                                                          *
* linkReplaceMsg   (**) 일반 링크의 대체 텍스트            *
*                  빈 문자열일 경우 대체하지 않음          *
*                  -> "대체할텍스트"                       *
*                                                          *
* chzzkChannelId   (***) 구독할 치지직 채널 ID             *
*                  채널 URL에서 확인 가능                  *
*                                                          *
* chzzkAuthServerUrl (***) chzzk-chat-server 주소          *
*                  기본값: "http://localhost:3000"         *
*                                                          *
* retryInterval    접속이 끊겼을 때 재접속 시도 시간 간격  *
*                  초                                      *
*                                                          *
* muteUser         (**) 특정 유저의 메세지를 표시하지 않기 *
*                  ["닉네임", "..."]                       *
*                                                          *
* commands         (*) 채팅방 명령어를 사용                *
*                  [{exe:"명령1", msg:"입력 명령어1"},     *
*                   {exe:"명령2", msg:"입력 명령어2"}]     *
*                  정규표현식을 알아야 사용할만 함         *
*                  기본 설정 이용 권장                     *
*                                                          *
* replaceMsgs      (*) 특정 텍스트를 변환하기              *
*                  [{orig:"이전1", to:"이후1"},            *
*                   {orig:"이전2", to:"이후2"}]            *
*                  정규표현식을 알아야 사용할만 함         *
*                                                          *
*                  nick:"영어아이디"를 포함하면            *
*                  특정유저의 메세지만 변환할 수 있음      *
*                                                          *
